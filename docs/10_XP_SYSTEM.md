# XP System

## Purpose

This document defines the Experience Points (XP) and leveling system for LinkUp V5. XP rewards user engagement and achievement while providing tangible benefits through level unlocks.

## Scope

- XP earning sources
- Level thresholds
- Level benefits
- Progress tracking
- XP display

## XP Philosophy

### Why XP?

XP serves several purposes:
1. **Recognition**: Acknowledge valuable contributions
2. **Motivation**: Incentivize positive behaviors
3. **Progress**: Show growth over time
4. **Unlocks**: Gate premium features by level

### What XP Is Not

- A currency (cannot be traded or spent)
- A way to buy advantages
- An engagement metric to maximize
- A reason to create low-quality events

## XP Rules

### Earning Rules

| Rule | Description |
|------|-------------|
| XP Never Decreases | Once earned, XP is permanent |
| XP Never Expire | Valid for lifetime |
| One-Time Bonuses | Only awarded once |
| Recurring Actions | Awarded each time |
| No XP Farming | Duplicate actions not counted |

### XP Limitations

| Action | Limit |
|--------|-------|
| Event XP per day | 500 |
| Rating XP per week | 200 |
| Report XP per week | 50 |

### XP Recovery

| Scenario | XP Recovery |
|----------|-------------|
| Event cancelled by organizer | Full event XP restored |
| Duplicate XP awarded (bug) | Corrected automatically |
| Fraudulent XP | Revoked, trust impact |

## Progress Tracking

### Progress Display

**In-App Locations:**
- Profile page: Level badge, XP total, progress bar
- Settings: Full XP history
- Notifications: Level-up celebrations

**Progress Bar:**
```
Level 5 → Level 6
[████████░░░░░░░░░░░░] 2,000 / 4,000 XP

```

### XP History

**Viewable Data:**
| Field | Description |
|-------|-------------|
| Date | When XP was earned |
| Action | What earned the XP |
| Amount | XP gained |
| Source | Event/meeting/etc |

**Privacy:**
- XP history private to user
- Other users see only level
- No leaderboard (by design)

## Edge Cases

### Rapid Leveling
- Multiple level-ups processed sequentially
- Each level celebrated separately
- Stacked unlocks shown together

### XP Disputes
- Appeal within 30 days
- Support reviews manually
- XP restored if valid claim

### Account Deletion
- XP permanently lost
- Cannot transfer to another account
- Level does not transfer
